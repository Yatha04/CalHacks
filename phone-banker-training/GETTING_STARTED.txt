================================================================================
  QUICK START (3 STEPS)
================================================================================

1. INSTALL DEPENDENCIES
   ----------------------
   cd phone-banker-training
   npm install

2. CONFIGURE API KEYS
   -------------------
   Edit .env.local file with your credentials:
   
   NEXT_PUBLIC_VAPI_PUBLIC_KEY=your_vapi_key_here
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url_here
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_key_here

   Get Vapi key: https://dashboard.vapi.ai
   Get Supabase keys: https://supabase.com (run DATABASE_SCHEMA.sql)

3. RUN THE APP
   ------------
   npm run dev

   Open: http://localhost:3000
